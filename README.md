# Udemy-Local-Git-Clone

This project is a simple Udemy Home Page Clone built using HTML5 and CSS3. The goal of this project was to practice front-end web development fundamentals by recreating the layout and design of Udemy's landing page.

Features Navigation bar with menu items Search bar design Course categories section Featured courses/cards layout Clean and modern UI inspired by Udemy Built using only HTML and basic CSS

Technologies Used HTML5 CSS3 Purpose

This project was created as a learning exercise to improve my understanding of:

HTML page structure CSS styling and layouts Flexbox positioning Responsive web design concepts UI cloning and design implementation

Note This is a front-end design project that replicates only the Udemy Home Screen for educational purposes. No backend functionality, authentication, or course management features are included.
